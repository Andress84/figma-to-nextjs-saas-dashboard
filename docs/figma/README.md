# Figma Resources

No editable Figma source files are currently included in this repository.

The approved design references and implementation handoff are stored in:

- [`/design`](../../design/)
- [`/design/README.md`](../../design/README.md)

The `design/` directory contains the approved screens, product specification,
component map, design tokens, responsive rules, fictional product data,
prototype reference, and visual QA requirements for the Next.js implementation.

## Reserved for future Figma UI Kit

This directory is intentionally kept as a placeholder for future Figma
artifacts related to the Subtera UI Kit and design system.

Possible future additions may include:

- a read-only Figma project link;
- reusable component documentation;
- variables and design-token mappings;
- component states and variants;
- responsive layout specifications;
- exported UI Kit previews;
- implementation notes connecting Figma components to React components.

Do not duplicate approved screenshots or handoff documents from `/design`
inside this directory.

Until editable Figma resources are added, `/design` remains the source of truth
for the approved product interface.
